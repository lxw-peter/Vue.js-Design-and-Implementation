# 内建组件和模块

## KeepAlive 组件的实现原理

> KeepAlive 借鉴于 HTTP 协议，作用是避免组件频繁地销毁、重建

### 使用场景

```html
<template>
  <!-- 使用 KeepAlive 组件包裹 -->
  <KeepAlive>
    <Tab v-if="currentTab ==== 1"></Tab>
    <Tab v-if="currentTab ==== 2"></Tab>
    <Tab v-if="currentTab ==== 3"></Tab>
  </KeepAlive>
</template>
```

### 卸载和挂载的过程

```shell
              |   |-- 页面 --|  
              |   |   ---   |   ‘卸载‘    --------    
 deactivated  |   |  |组件|--|---------> |隐藏容器|   
              |   |   ---   |            --------  
              |   |_________|  
--------------|-----------------------------
              |
              |   |-- 页面 --|     
  activated   |   |   ---   |  重新‘挂载’ --------   
              |   |  |组件|--|---------> |隐藏容器|
              |   |   ---   |           ---------    
              |   |_________|         

```

## Teleport 组件的实现原理

### 作用

将指定内容渲染到特定容器中，以解除 DOM 层级的限制

## Transition 组件的实现原理

### 核心原理

- DOM 元素被挂载，将动效附加到 DOM 元素上；
- DOM 元素被卸载时，等到附加到该元素的动效执行完毕再卸载。
